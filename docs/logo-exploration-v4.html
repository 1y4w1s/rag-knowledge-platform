跳出「岸/灯塔」约束 · 原生概念重探

唯一硬约束：赤陶橙 #CB6B3D。以下 4 个方向均从「RAG 知识库 / 引用溯源 / 可信协作」的产品本质长出来，互不雷同，且零外来风格词汇。

为什么不再锁 B：「岸/灯塔」是我从产品名「知岸」自己推导的隐喻，你从未指定它。把它当唯一解等于提前缴械。v4 把隐喻层也打开——只保留颜色这把尺子，让概念从产品功能本身生长。四张参考图的 6 条法则 仍是评判尺（形式规则 / 第二眼奖励 / 视觉重量 / 产品绑定 / 缩放可读 / 色彩文化适配），但不再借用它们的「长相」。
Concept D1

锚 · Anchor

「知岸」天然成对的航海符号：岸 + 锚。锚 = 停靠、稳定、有据可依——正好对应 RAG 的「引用溯源、答案可锚定到原文」。

Concept D4

棱 · Prism

棱镜把一束杂光折射成有序光谱 = 把零散文档「提炼」成可信答案。仅用赤陶的透明度变化模拟「光谱」，不引入第二种颜色。
