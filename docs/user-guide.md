# 睿阁 — 用户手册

> 面向终端用户
> 更新日期：2026-07-17

---

## 1. 快速开始

### 什么是睿阁？

睿阁是一个企业知识库平台。你可以上传公司内部的文档（PDF、Word、Excel、Markdown 等），然后像聊天一样向知识库提问，系统会从文档中检索相关信息并给出带引用的回答。

### 注册账号

1. 访问系统首页，点击「注册」
2. 选择账号类型：
   - **个人版**：单人使用
   - **企业版**：团队使用（创建组织后可以邀请成员）
3. 填写邮箱、用户名、密码，完成注册

---

## 2. 资料库管理

### 创建资料库

1. 登录后进入「资料库」页面
2. 点击「创建资料库」
3. 输入名称，选择所属部门（企业版）

### 上传文档

支持的文件类型：

| 格式 | 说明 |
|------|------|
| PDF | 文本版 + 扫描件 OCR |
| Word (.docx) | 含标题层级解析 |
| Excel (.xlsx) | 每张表格作为一个独立片段 |
| PowerPoint (.pptx) | 每页幻灯片文本提取 |
| Markdown (.md) | |
| 纯文本 (.txt) | |

上传方式：

1. 进入资料库详情页
2. 点击「上传文档」
3. 选择文件（可多选）
4. 等待系统自动处理（ingestion 状态：排队中→处理中→完成）

### 文档版本

同名文档再次上传时，不会覆盖已有文档，而是创建新版本。可以在文档详情页查看版本历史。

## 3. 对话

### 单库对话

1. 进入资料库详情页
2. 点击「对话」按钮
3. 输入问题，按回车
4. 系统会从该资料库中检索相关信息并给出回答

### 跨库对话

1. 点击顶部导航「智能问答」
2. 选择需要搜索的一个或多个资料库
3. 输入问题
4. 系统会从所选资料库中综合检索

### 理解回答

每个回答底部会显示引用来源：

```
回答内容...
──
来源：员工手册.pdf → 第三章 考勤制度 → 3.1 加班
```

点击引用可以跳转到文档中的对应位置。

### 反馈

每次对话后，可以对回答进行 👍 或 👎 评价。你的反馈会帮助改进系统。

## 4. 组织管理（企业版）

### 邀请成员

1. 进入「管理」→「成员」
2. 点击「邀请成员」
3. 复制邀请链接或邀请码，发送给团队成员
4. 被邀请人通过链接或码完成注册

### 成员角色

| 角色 | 权限 |
|------|------|
| Owner | 全部权限，包括转让所有权 |
| Admin | 管理成员、创建资料库、删除资料库 |
| Member | 查看和对话资料库（不能创建/删除）|
| 自定义角色 | Admin 可以创建自定义角色，按资料库粒度设置权限 |

### 部门管理

1. 进入「管理」→「部门」
2. 创建部门树结构
3. 将资料库分配给部门
4. 部门成员只能看到本部门的资料库

## 5. 账号设置

### 修改密码

1. 点击右上角头像 → 「账号设置」
2. 输入当前密码和新密码
3. 保存

### 忘记密码

1. 在登录页点击「忘记密码」
2. 输入注册邮箱
3. 查收重置邮件，点击链接设置新密码

## 6. 常见问题

**Q: 上传后文档一直显示「处理中」？**
A: 文档需要经过解析→切片→向量化的过程。小文件通常几秒完成，大文件可能需要 1-2 分钟。如果长时间未完成，请检查文档是否为不支持的类型。

**Q: 回答不准确怎么办？**
A: 可以尝试换一种问法。对话底部的 👎 反馈可以帮助我们追踪问题。

**Q: 如何删除文档？**
A: 在资料库详情页的文档列表中，点击文档右侧的删除按钮。删除后文档会进入回收站，可以从回收站彻底删除。

**Q: 同名文档上传会覆盖吗？**
A: 不会覆盖。系统会自动创建新版本，旧版本可以在版本历史中查看和回滚。
