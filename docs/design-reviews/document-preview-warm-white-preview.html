2026 年第二季度财报分析

财务知识库 · 最后更新 2026-07-08 · 共 12 页

一、经营概览

本季度公司实现营业收入 4.82 亿元，同比增长 23.6%；归属于母公司净利润 0.91 亿元，净利率提升至 18.9%。现金及等价物余额为 6.3 亿元，流动性充裕。

二、分业务板块

智能检索业务贡献营收 2.74 亿元（占比 56.8%），企业知识库订阅收入 1.55 亿元，专业服务及其他 0.53 亿元。订阅类收入占比持续上升至 64%，收入结构更稳健。

三、成本与费用

研发投入 1.18 亿元，占营收 24.5%；销售及管理费用合计 1.02 亿元，费用率较上季下降 2.1 个百分点，规模效应初步显现。

四、展望

下半年重点推进多模态检索与私有化部署，预计全年营收区间 10.5–11.2 亿元。相关风险提示详见第四节附注。

# 产品需求文档（PRD）· 知岸企业知识库
> 状态：评审中 · 负责人：产品组 · 最后更新 2026-07-05

## 1. 背景
企业内部的文档散落在网盘、邮件与 IM 中，检索效率低、权限混乱。
知识库面向"企业级 RAG 检索"场景，统一接入与权限治理。

## 2. 目标用户
- 知识库管理员：负责上传、切块、权限与审计。
- 普通成员：在资料库内提问，引用可溯源。

## 3. 核心功能
1. 多格式接入（PDF / Word / Markdown / 图片 OCR）。
2. 切片与向量化，支持"处理中 / 完成 / 失败"状态可视化。
3. 对话引用溯源，每条回答可展开原文片段。

## 4. 非目标
- 不在此版本支持实时协同编辑。
- 不支持音视频内嵌播放。
