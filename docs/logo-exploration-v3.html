概念全部来自品牌自身：岸 / 灯塔 / 赤陶橙 / 知识之屿。 四张参考只作为「质量标尺」（形式规则、视觉重量、第二眼奖励、缩放可读、产品绑定）， 不作为「风格来源」。本版没有任何印章 / 罗盘 / 字母标等外来词汇。

方法论修正：参考图 = 质量标杆（告诉你好 LOGO 长什么样），品牌概念 = 原料（岸/灯塔/赤陶/RAG）。 我只用前者的「尺子」量后者的「料」，不把前者的「样子」搬过来。 v2 的 D(印章)/F(罗盘)/G(字母标) 已废弃——它们借了参考的视觉词汇，违反应有的「品牌原生」原则。

这次的边界：三版都是「岸 + 灯塔/光 + 赤陶」这一条品牌主线，只换形式装置（光锥当引用链 / 对称峰 / 上升光束）。 没有任何外来风格词汇。你之前说 B 还行——这就是 B 被那把「质量尺」量过之后的三个样子。

选哪个看你要什么气质： B1 灯塔溯源 = 最贴产品（光=引用）、最有记忆钩子； B2 岸峰 = 最稳重极简、favicon 最保险； B3 升梁 = 最现代轻盈、最「编辑感」。 选定后我导出矢量资产 + 品牌规范段落。
