## 三、鲁棒性评估（Robustness）

### 3.1 极端场景测试结果（11 项全通过）

| 场景 | 覆盖 | 结果 |
|------|------|------|
| LLM 下游 502 | stream_deepseek 异常传播 | ✅ |
| LLM 5xx HTTP 端到端 | 服务不崩 | ✅ |
| Embedding 503 | embed_texts 抛异常 | ✅ |
| Embedding 5xx Ingestion | 文档 failed 状态 | ✅ |
| DB 连接断开 | 3 个 API 返回异常 | ✅ |
| 存储写入 OSError | 不落脏数据 | ✅ |
| 预览文件不可读 | 返回 404/410 | ✅ |
| Ingestion 读取失败 | failed + 中文错误 | ✅ |
| 登录限流 5次/15min | → 429 | ✅ |
| IP 级限流 20次/5min | → 429 | ✅ |
| 渐进锁定期 1m→5m→15m→1h | 递增等待 | ✅ |

### 3.2 降级策略覆盖

| 故障 | 降级行为 |
|------|----------|
| LLM 调用失败 | 异常传播，不静默吞掉 |
| Embedding 失败 | 文档标记为 failed |
| DB 不可用 | → 503 Service Unavailable |
| 存储不可用 | → 500 + 中文提示 |
| 检索结果为空 | query rewrite 重试一次，仍空→拒答 |

**评级**: ⭐⭐⭐⭐（4/5）
