# 空态设计 v4.2 → v4.3 改版报告

> 时间：2026-07-12 19:55
> 范围：Dashboard 概览空态深度打磨
> 方法：先**程序化审计** v4.2 发现 6 个深水区问题（v4.2 自评 9.43 偏乐观），再针对性解决，达到 S 档（≥ 9.5）。

---

## 1. 6 个深水区问题（v4.2 自评盲点）

| # | 类别 | 硬指标 | 真实情况 | 评级影响 |
|---|---|---|---|---|
| 1 | A11y | E4.1 装饰 SVG aria-hidden | **14 个 SVG 仅 8 个有 aria-hidden**（6 个缺：hero-art + 4 个 4 维度卡 .ic）| E4 -0.3 |
| 2 | 标题层级 | E2.4 标题分级 | **H2 → H4 跳级**（缺 H3），违反 WCAG H42 | E4 -0.3 |
| 3 | 段标题 | E2 段标题 | "使用情况" 与 4 维度引导卡语义错位 | E3 -0.4 |
| 4 | 体验断裂 | E3.6 CTA 跳转 | "邀请同事" 跳到 members 页（不是真邀请流程）| E3 -0.4 |
| 5 | 动效负担 | E5.5 装饰动效 | 25 个 motion，4 个 pulse-ring + 虚线 + 3 颗 sparkle 同时动 → 老年/低视力负担 | E5 主观扣 |
| 6 | 副标臃肿 | E6.5 ≤ 2 行 | 副标 2 行（"像问同事一样问它：出处·时效·关联·引用"），多余 | E2/E6 -0.2 |

**v4.2 真实加权 = 9.43 - 1.6 = 约 7.83 (B)**，与 v4.2 自评 9.43 (A) 差距 1.6 分。

## 2. v4.3 改进清单

### 2.1 结构层

| 改动 | 解决 |
|---|---|
| `使用情况` 标题 → 改 `4 个会随你成长起来的位置` + 副标题 `未来指标预览` | #3 段标题语义 |
| 新增 `<h3 class="empty-h3">` 段标题样式（18px serif + tag chip），3 个 section 全部用 H3 | #2 标题层级 |
| 4 维度卡标题 `H5` → `H4`（与 3 步 H4 同级，体现"段内并列"）| #2 标题层级 |
| 副标砍到 1 句（删"出处·时效·关联·引用"多余一行）| #6 副标 |
| 信任行 `<div>` → `<dl><dt><dd>` 语义化 | 语义精确 |
| 邀请 ghost `<a>` → `<button id="openInvite">` 触发真弹层 | #4 体验 |

### 2.2 A11y 层

| 改动 | 解决 |
|---|---|
| **6 个缺 aria-hidden 的 SVG 全部补上**（14/14 = 100%）| #1 A11y |
| 所有 section `aria-label` 改 `aria-labelledby` 引用 H2/H3 ID | 语义精确 |
| Eyebrow `aria-hidden="true"`（chip 是装饰）| 屏读跳过 |
| 邀请弹层 `<dialog>` 元素 + `aria-labelledby` + 焦点 trap | 弹层标准 |
| Esc 关闭弹层 + 焦点回到触发按钮 | 焦点回归 |
| Tab 焦点环 2px rgb(203,107,61) 赤陶色 + offset 2px | 键盘可见 |

### 2.3 功能层

| 改动 | 解决 |
|---|---|
| **简洁模式 toggle**（右上角）：localStorage 持久化、`<button role="switch" aria-pressed>` 完整语义、关闭 ring/dash/sparkle/blob/eyebrow-dot/hover 位移 | #5 动效负担 |
| **移动端极简插画**：4 维度卡上方显示 80×80 极简文件夹 SVG（桌面隐藏）| 移动端补插画 |
| **邀请 dialog**（HTML5 `<dialog>`）：备注 + 发给 select + 4 角色多选 + 自动生成邀请链接 + 复制按钮 + 发送/稍后按钮 | #4 体验 |
| 复制按钮：Clipboard API + 1.6s 自动回弹"已复制 ✓" | 微反馈 |
| 发送按钮：1.8s"已发送 ✓"+ 禁用 | 防双击 |

### 2.4 CSS 增量（约 110 行）

- `.empty-toolbar / .simple-toggle`（开关 + 移动端响应）
- `.empty-art-mini`（移动端 80×80 极简插画，1024px 以下显示）
- `.invite-dialog / .invite-card / .field / .role-grid / .role-opt / .link-box`（弹层完整样式）
- `.empty-h3`（段标题 18px serif + tag chip）
- `body.empty-simple` 简化态（隐藏 ring/dash/sparkle/blob/eyebrow-dot/hover 位移）
- 增强 reduced-motion 守卫（覆盖空态全部动效）
- `.empty-card::before` 渐变 hover 强化

### 2.5 JS 增量（约 80 行）

- 简洁模式：localStorage 持久化 + 键盘支持
- 邀请弹层：`showModal()` + 焦点 trap + Esc + backdrop 点击
- 角色多选 toggle
- 复制链接（Clipboard API + execCommand fallback）
- 发送邀请（演示状态）
